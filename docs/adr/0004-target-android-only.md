# Target Android Only for Initial PWA Migration

We are explicitly ignoring iOS-specific PWA constraints and features (such as `apple-touch-icon` requirements and iOS-specific Add-to-Home-Screen notification rules) during this migration.

The application is currently only used in an Android setting. Attempting to support iOS adds unnecessary scope and complexity regarding Web Push limitations. We will focus our efforts purely on delivering a high-quality Android PWA experience, and address iOS compatibility in the future if the user base expands.
